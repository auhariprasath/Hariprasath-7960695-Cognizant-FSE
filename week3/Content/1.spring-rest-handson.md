# Hands-on 1
## Create a Spring Web Project using Maven

### Objective
Create a Spring Boot Web application using Maven and understand the project structure.

### Technologies Used
- Java 21
- Spring Boot 4.x
- Maven
- Spring Web
- Spring Boot DevTools

### Steps Performed

1. Generated project using Spring Initializr
2. Group Id: `com.cognizant`
3. Artifact Id: `spring-learn`
4. Added Dependencies
   - Spring Web
   - Spring Boot DevTools
5. Imported project into Eclipse
6. Built project using Maven

```bash
mvn clean package
